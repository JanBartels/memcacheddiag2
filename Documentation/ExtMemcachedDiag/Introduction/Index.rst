Introduction
------------

The extension implements a diagnosis for memcached-servers that
are configured as backends for the caching-framework.

Memcached is a free and open source, high-performance, distributed
memory object caching system.
To learn more about Memcached, go to
`http://memcached.org <http://memcached.org/>`_ .
